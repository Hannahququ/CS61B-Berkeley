## First Java Program

### 1. Hello World & Hello Numbers

```java
public class HelloWorld {
	public static void main(String[] args) {
		System.out.println("hello world");
	}
}
```

```java
public class HelloNumbers {
	public static void main(String[] args) {
		int x = 0;
		while (x < 10) {
			System.out.println(x);
			x = x + 1;
		}

		x = "horse";
	}
}
```
